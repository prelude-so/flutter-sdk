# Change Log

Prelude Flutter SDK Change Log

## [0.2.1] - 2026-01-22

* Update native Android SDK to version 0.2.5. Adds network-bound DNS resolver for improved Silent Network Authentication (SNA) reliability.

## [0.2.0] - 2025-12-17

* Added example that shows how to use the SDK.
* Updated to use the latest native SDKs versions.
* Updated documentation.

## [0.1.0] - 2025-12-05

* Initial release of the Prelude Flutter SDK. This SDK wraps the native Android and iOS Prelude SDKs providing an easy-to-use interface for Flutter applications that integrate Prelude's services.
